## Installation

1. Firstly check out the code from git Hub repo

2. install all packages
   - npm install
3. Run the application
   - npm start

## Run Test Cases

- npm run test

## Tech Stack

1. React
2. TypeScript: A superset of JavaScript that adds static typing.
3. React Router: A routing library for React.
4. Redux: library for managing and centralizing application state.
5. Redux-Toolkit
6. Redux Saga: A middleware library for managing side effects in Redux applications.
7. Axios: Libraries for making HTTP requests from the browser.
8. Jest: A JavaScript testing framework.
9. React Testing Library: for unit testing of components
